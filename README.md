# 全国大学生竞赛 DDL 网

一个收录全国大学生竞赛白名单赛事的倒计时页面。静态部署即可运行，适合放在 Vercel / Netlify 等平台。

## 功能
- 统一展示白名单赛事，按“名单顺序”或“报名截止”排序。
- 分类（左侧抽屉）、级别筛选、关键词搜索、只看未截止开关。
- 卡片展示报名/开赛节点、官网、倒计时、时间轴。
- 顶栏显示数据更新时间；赞助弹窗（`pay.JPG` 收款码）。
- 自适应移动端，深色高端风格。

## 使用
1. 安装依赖：无需。
2. 本地预览：直接用浏览器打开 `index.html`。
3. 部署到 Vercel：导入仓库，框架选 “Other”，无需构建命令，Output 目录留空。

## 目录
- `index.html`：页面结构
- `styles.css`：样式
- `app.js`：数据和交互逻辑
- `pay.JPG`：赞助收款码

## 数据更新
- 赛事数据在 `app.js` 的 `detailedCompetitions` 和 `whitelistBase` 中维护。
- 时间未发布时备注“待发布”并标注最近核对日期（2025-12-07）。
